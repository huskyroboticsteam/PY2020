\doxysubsection{Detailed Description}
Class representing an AR tag detected in an image. 

This class is simply used as a way to represent the result of a detection, and can provide location and orientation information. You should likely never need to construct a \mbox{\hyperlink{classAR_1_1Tag}{Tag}} yourself; this is done internally by the detection code. All tags are immutable after construction.

Gets the translation vector of the tag. This vector defines the translation necessary to transform points from the object reference frame to the camera reference frame. For the center of the tag (0,0,0), this vector is in effect the 3D coordinates of the tag relative to the camera.

This function will return real world units; these are dependent upon the camera parameters you are currently using.
